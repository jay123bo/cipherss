  HOW TO GET STARTED:
  ═══════════════════

  STEP 1 — Install Python (if you haven't already)
    Go to: https://www.python.org/downloads
    Download and install it.
    IMPORTANT: Check "Add Python to PATH" during install!

  STEP 2 — Create your training data
    Create a file called data.txt in this folder.
    Paste in lots of text — books, articles, anything.
    The more text, the smarter your AI.
    
    Free books to copy from: https://www.gutenberg.org
    (Search a book, open it, press Ctrl+A then Ctrl+C, paste into data.txt)

  STEP 3 — Run SETUP.bat
    Double-click SETUP.bat
    It installs everything and trains your AI automatically.
    This takes 30min–2 hours. Leave it running!

  STEP 4 — Chat with your AI
    Double-click "CHAT WITH MY AI.bat"
    Type anything and press Enter!

  ════════════════════════════════════════════════

  MAKING YOUR AI SMARTER:
    Add more text to data.txt and run SETUP.bat again.
    The more you train it, the better it gets.

  YOUR AI IS SAVED IN: my_ai.pt
    This file is your AI's brain. Don't delete it!
